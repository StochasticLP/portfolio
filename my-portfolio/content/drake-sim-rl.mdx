# Reinforcement Learning

Reinforcement Learning (RL) involves training an agent to make decisions by interacting with an environment.

## The Agent

Our RL agent has been trained to swing up and balance the pole. Unlike LQR, which requires a mathematical model, RL learns from trial and error.
