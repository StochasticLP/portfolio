# Portfolio Website

A portfolio website built with [Next.js](https://nextjs.org/), [Vercel](https://vercel.com/), [Tailwind CSS](https://tailwindcss.com/), and [Drake](https://drake.mit.edu/).

## Technologies Used

- Framework: [Next.js](https://nextjs.org/)
- Styling: [Tailwind CSS](https://tailwindcss.com/)
- Analytics: [Vercel Web Analytics](https://vercel.com/docs/speed-insights) and [Speed Insights](https://vercel.com/docs/speed-insights)
- Deployment: [Vercel](https://vercel.com/)
- Client Side Simulation: [Matter.js](https://brm.io/matter-js/)
- Server Side Simulation: [Drake](https://drake.mit.edu/)
- Visualizations: [Meshcat](https://github.com/ethz-pbs/meshcat)
